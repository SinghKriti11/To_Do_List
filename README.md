##📝 To-Do List 

A modern and interactive To-Do List application built using HTML, CSS, and JavaScript. 
This app allows users to manage daily tasks efficiently with features like task creation, deletion, completion tracking, and deadlines.

##  Features

*  Add new tasks
*  Delete tasks
*  Mark tasks as completed
*  Set deadlines for tasks
*  Displays time remaining (in hours)
*  Data persistence using Local Storage
*  Automatic sorting (unfinished tasks on top)
