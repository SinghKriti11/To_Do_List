# 📝 To-Do List 

A modern and interactive **To-Do List application** built using HTML, CSS, and JavaScript. This app allows users to manage daily tasks efficiently with features like task creation, deletion, completion tracking, and deadlines.

## 🚀 Features

*  Add new tasks
*  Delete tasks
*  Mark tasks as completed
*  Set deadlines for tasks
*  Displays time remaining (in hours)
*  Data persistence using Local Storage
*  Automatic sorting (unfinished tasks on top)
*  Modern UI with glassmorphism design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Flexbox, Glass UI, Responsive layout)
* **JavaScript ** – Functionality & DOM manipulation

## ⚙️ How It Works

1. Enter a task in the input field
2. Optionally select a deadline
3. Click **Add**
4. Click on a task to mark it as completed
5. Click ❌ to delete a task
6. Tasks are saved automatically in browser storage
